# 케이시엠교육사회적협동조합 홈페이지

학교와 지역이 함께 만드는 교육 공동체를 위한 사회적협동조합 공식 홈페이지입니다.

## 🎯 프로젝트 개요

케이시엠교육사회적협동조합은 협동조합기본법에 따라 설립된 사회적협동조합으로, 학생·학부모·교직원·지역주민이 함께 참여하여 더 나은 교육 환경을 만들어가는 교육 공동체입니다. 본 홈페이지는 조합의 투명한 운영과 조합원 소통, 그리고 지역사회와의 협력을 위해 제작되었습니다.

## ✨ 주요 기능

- 총회 참석 방법 안내

### 2. 기술적 특징

#### 프론트엔드
- **HTML5**: 시맨틱 마크업 구조
- **CSS3**: 반응형 디자인, Flexbox/Grid 레이아웃
- **JavaScript (ES6+)**: 바닐라 JS로 인터랙션 구현
- **Font Awesome**: 아이콘 라이브러리
- **Google Fonts**: Noto Sans KR 폰트

#### 백엔드 연동
- **RESTful API**: Tables API 활용
- **데이터베이스 테이블**: `posts` (게시글 관리)
- **CRUD 기능**: Create, Read, Update, Delete 구현
- **샘플 데이터**: 5개 게시글 사전 등록

#### 반응형 디자인
- 데스크톱 (1200px+)
- 태블릿 (768px ~ 1199px)
- 모바일 (~ 767px)
- 햄버거 메뉴 (모바일)

## 🗄️ 데이터베이스 스키마

### posts 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 게시글 ID (자동생성) |
| title | text | 게시글 제목 |
| content | rich_text | 게시글 내용 |
| author | text | 작성자 이름 |
| category | text | 카테고리 (공지사항, 활동소식, 자유게시판, 문의사항) |
| status | text | 게시 상태 (pending, approved, rejected) |
| views | number | 조회수 |
| created_at | datetime | 작성일시 |

### 샘플 데이터 (5개)
1. **공지사항**: "2025년 제13차 정기총회 개최 안내" (조회수 125)
2. **활동소식**: "봄 교육프로그램 운영 결과 보고" (조회수 87)
3. **자유게시판**: "조합원 등산 모임 후기" (조회수 42)
4. **문의사항**: "후원 방법 문의드립니다" (조회수 38)
5. **공지사항**: "장학금 지원 대상자 모집 안내" (조회수 156)

## 🔗 RESTful API 사용

### API 엔드포인트
```javascript
// 게시글 목록 조회 (페이지네이션, 카테고리 필터, 검색)
GET /tables/posts?page=1&limit=10&category=공지사항&status=approved&sort=-created_at

// 게시글 상세 조회
GET /tables/posts/{id}

// 게시글 작성
POST /tables/posts
Body: { title, content, author, category, status: 'pending', views: 0 }

// 조회수 증가
PATCH /tables/posts/{id}
Body: { views: 1 }
```
